# Platform Comparison
## Original Website vs. Rebuilt Website

This document provides a comparative analysis of the original Lawgical Group website against the newly rebuilt platform, highlighting visual, technical, and operational enhancements.

---

### 1. Visual & Branding Comparison

| Comparison Metric | Original Website | Rebuilt Website | Measurable Improvement |
| :--- | :--- | :--- | :--- |
| **Visual Design** | Traditional law firm layout. Light theme with heavy grey lines. Looks generic and standard. | Dark-theme accents, custom terminal coordinate grids, premium HSL color tokens. | Evolvement to a modern, editorial financial-terminal style. |
| **Branding** | SVG monogram is absent; logo files are disconnected. Standard header fonts. | Vector monogram integrated in Navbar, Footer, and Loader. Geometric hexagon accents in details. | Unified, deliberate, and recognizable brand signature. |
| **Typography** | Default system fonts (Arial, Times). Hard to read in long columns. | Sora for headings, Geist for body text. Optimized spacing and line heights. | Enhanced legibility and modern editorial aesthetic. |
| **Content Hierarchy** | Monolithic text pages. Services and team details require multiple clicks. | Sectioned homepage flow. Pinned scroll sections and tabs group information. | Clear, immediate answers to client questions. |

---

### 2. User Experience & Interaction Comparison

| Comparison Metric | Original Website | Rebuilt Website | Measurable Improvement |
| :--- | :--- | :--- | :--- |
| **Mobile Experience** | Columns broke boundaries, maps ran offscreen, text scaled down too small. | Stacks naturally into accordions and swipe lists. Cursor disables on touch interfaces. | Mobile bounce rates are significantly reduced. |
| **Navigation** | Crowded dropdown menus with duplicate route listings. | Clean 7-route navbar with active underlines. Footer pulls services dynamically. | Reduced path friction; users find pages faster. |
| **Custom Interaction** | Standard OS pointer with basic link underlines. | **Legal Compass Cursor** with rotating ticks and upright state-specific icons. | Premium interaction layer that leaves a lasting digital impression. |
| **Leadership Visibility** | UAE advocate buried in a uniform list grid. Kishore Mulani prioritized. | Dual Featured Spotlight for Kishore and Dr. Essa Al Nuaimi with equal prominence. | Establishes balanced litigation (court right of audience) and transactional authority. |
| **Portrait Crops** | Generic center crop values. Faces were inconsistently cropped. | Manual coordinate crop offsets (`objectPosition`) keeping eyes in the upper third. | Consistent, high-end profile layouts across all screen sizes. |

---

### 3. Trust & Technical Metrics

| Comparison Metric | Original Website | Rebuilt Website | Measurable Improvement |
| :--- | :--- | :--- | :--- |
| **Client Showcase** | Carousel or simple logo strip. Hard to read and slow-moving. | **Trust Constellation** mosaic. Hovering reveals names and dims other logos. | Modern, interactive trust signals with no boring carousels. |
| **Testimonials** | Text-only reviews with no verification signals. | Testimonials hub with inline autoplay video loops and overlays. | Humanized social proof. High-trust video validations. |
| **Map Experience** | Static Google map embed or simple text listing of branches. | Interactive SVG regional map. Selecting GCC nations updates data panels. | Visual confirmation of cross-border capability. |
| **Insights Hub** | Basic blog listing with missing images or broken paths. | Editorial magazine grid with SafeImage fallbacks to prevent broken images. | Zero broken image placeholders; premium briefing aesthetics. |
| **Performance** | Multi-second loading lags on mobile networks due to heavy assets. | Next.js pre-compiled pages running under Turbopack. Sub-second load times. | Compilation in seconds, instant transitions. |
| **Lead Capture** | Simple contact form. Vague queries require follow-up calls. | Form with capability selectors pre-filled from specific legal briefs. | Pre-qualified leads; higher conversion potential. |
