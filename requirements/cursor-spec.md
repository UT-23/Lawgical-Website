# Cursor Specification

## Cursor Theme

Name:
Legal Gavel Cursor

Purpose:
Create a distinctive premium interaction signature.

---

## Default State

Appearance:
- Minimal outlined gavel
- Thin stroke
- Elegant proportions

Behavior:
- Follow cursor smoothly
- No lag
- GPU accelerated

---

## Link Hover

Behavior:
- Rotate 10 degrees
- Slight glow

Duration:
200ms

---

## Button Hover

Behavior:
- Scale to 1.15x
- Accent highlight

Duration:
200ms

---

## Attorney Card Hover

Behavior:
- Tooltip reveal
- Slight rotation

Tooltip Examples:
- View Profile
- Learn More

---

## Loading State

Behavior:
- Subtle rotation
- Reduced opacity

---

## Accessibility

Rules:
- Allow disabling custom cursor
- Respect touch devices
- Do not replace native cursor on mobile

---

## Restrictions

Avoid:
- Cartoon styling
- Oversized cursor
- Aggressive motion
- Performance-heavy effects
