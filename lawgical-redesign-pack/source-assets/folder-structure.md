# Suggested Folder Structure for the Original Site Assets

```text
source-assets/
  logos/
  team/
  hero/
  services/
  industries/
  news/
  videos/
  documents/
  screenshots/
  icons/
  thumbnails/
```

## Notes
- Put all portraits in `team/`
- Put all homepage and section hero images in `hero/`
- Put all videos and looping motion clips in `videos/`
- Put all downloaded PDFs or brochures in `documents/`
- Put screenshots of each page and key state in `screenshots/`
