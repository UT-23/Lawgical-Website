# Asset Extraction Plan

## What to extract from the original site
1. Brand assets
   - main logo
   - alternate logo
   - favicon
   - icon set
2. People assets
   - portrait for every team member
   - group team images
3. Visual assets
   - homepage hero images
   - office images
   - service visuals
   - industry visuals
   - news/article thumbnails
   - contact/location visuals
4. Motion assets
   - videos
   - looping backgrounds
   - animated illustrations
5. Content assets
   - page copy
   - metadata
   - CTA labels
   - footer links
   - legal disclaimers
6. Technical assets
   - PDFs
   - embedded map references
   - downloadable resources

## Recommended extraction rule
Keep original filenames where possible, but normalize them for ease of use:
- lowercase
- hyphen-separated
- section-based naming
- no spaces

## Naming examples
- logo-primary.svg
- logo-mark.svg
- team-name-surname.jpg
- hero-dubai.jpg
- service-corporate-law.png
- news-featured-01.jpg
- video-home-loop.mp4

## Suggested output folders
- /logos
- /team
- /hero
- /services
- /industries
- /news
