# Media Map Template

For every extracted media file, record:

```json
{
  "filename": "team-01.jpg",
  "page": "team",
  "section": "leadership-grid",
  "type": "portrait",
  "usage": "team card avatar",
  "notes": "primary portrait, crop to 4:5"
}
```

## Suggested metadata fields
- filename
- source page
- section
- media type
- recommended crop
- fallback behavior
- accessibility alt text
- load priority
- usage priority
