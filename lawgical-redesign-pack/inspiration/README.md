# Inspiration Folder

This folder collects the redesign direction and the interaction system for the Lawgical Group revamp.

## What this is
A planning pack for a fully reimagined website that keeps the same content, but changes the experience completely:
- more motion
- more depth
- more storytelling
- stronger conversion moments
- premium legal-brand aesthetics
- modern interaction patterns inspired by the benchmark sites

## How to use
Use these files as the source material for Windsurf or Cursor. Keep the content, replace the presentation.
