# Interaction Library

## Purpose
This document defines the interaction vocabulary of the website.

Every interaction should communicate:
- trust
- authority
- precision
- professionalism

Never use motion for decoration only.

---

## Hero

Component:
ReactBits Shuffle Text

Purpose:
Create engagement without sacrificing trust.

Behavior:
- Rotate practice areas
- Shuffle animation
- Fade reveal

Timing:
4–5 seconds

---

## Practice Areas

Component:
KodyTechnolab Style Image Reveal Cards

Default:
- Large number
- Practice title
- Short description

Hover:
- Image reveal
- Overlay fade
- Content shift upward
- CTA reveal

Duration:
500ms

---

## Attorneys

Component:
Premium Hover Profile Cards

Hover:
- Image zoom
- Gold glow
- Bio reveal
- CTA reveal

Purpose:
Increase profile engagement.

---

## Contact

Component:
Segmented Service Chips

States:
- Default
- Hover
- Active

Behavior:
- Active border
- Soft glow
- Smooth transition

---

## Offices

Component:
Expanding Office Panels

Default:
- Collapsed vertical tabs

Hover:
- Expand width
- Reveal office image
- Reveal address
- Reveal contact details

Duration:
600ms

---

## Testimonials

Component:
Embla Carousel

Behavior:
- Auto play
- Pause on hover
- Keyboard accessible
- Smooth transitions

---

## Metrics

Component:
Animated Counters

Trigger:
Viewport enter

Duration:
1200ms

Examples:
- Years of experience
- Clients served
- Transactions advised

---

## Navigation

Component:
Premium Mega Menu

Behavior:
- Fade
- Slide
- Scale

Show:
- Image preview
- Description
- CTA

No abrupt appearance.
