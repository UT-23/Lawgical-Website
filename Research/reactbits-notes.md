# React Bits Notes for the Law Firm Site

## Why React Bits matters
React Bits is useful when the goal is to add polished motion without building every effect from scratch. It is especially helpful for text animation, scroll reveal, and subtle engagement layers.

## Best-fit components for a law firm
### 1. Shuffle Text
Use for:
- Hero headline
- Practice area lead-in
- Rotating value proposition

Why it fits:
- Signals modernity
- Creates attention without being noisy
- Works well with legal copy if used sparingly

### 2. Split / Reveal Text
Use for:
- Section headings
- Attorney bios
- About page hero statements

Why it fits:
- Elegant and controlled
- Feels premium and editorial

### 3. Animated counters
Use for:
- Case results
- Firm achievements
- Years of experience
- Client count

Why it fits:
- Adds credibility
- Reinforces authority

### 4. Scroll reveal effects
Use for:
- Section entrances
- Timeline milestones
- Value propositions
- Testimonial blocks

Why it fits:
- Makes the page feel alive
- Keeps long pages engaging

### 5. Gradient / glow text
Use only in small doses:
- Hero accent words
- One-line emphasis
- CTA emphasis

Avoid overuse:
- Too much glow looks startup-like
- Keep legal branding restrained

## Implementation guidance
- Use React Bits only where the motion serves comprehension or trust
- Prefer one motion idea per section
- Respect reduced motion preferences
- Keep animation timing in the 300ms to 800ms range for most interactions
- Avoid constantly looping motion unless it is extremely subtle

## Recommended usage order
1. Shuffle text in hero
2. Scroll reveal for content sections
3. Animated counters for proof points
4. Split text for premium section headings
